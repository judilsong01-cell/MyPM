# MyPME Investor Pack

Este diretorio concentra os materiais de apresentacao do MyPME para reunioes com potenciais investidores, socios e parceiros comerciais.

## Documentos

- `01_Resumo_Executivo.md` : visao de negocio em leitura rapida
- `02_Tese_de_Investimento.md` : porque este projeto pode escalar
- `03_Ficha_Tecnica_do_Produto.md` : stack, arquitetura e estado atual
- `04_Go_To_Market_e_Modelo_de_Negocio.md` : monetizacao e tracao inicial
- `05_Roadmap_e_Pedido_de_Investimento.md` : proximas fases e uso de capital
- `MyPME_Investor_Deck.html` : apresentacao visual pronta para abrir no browser

## Como usar

1. Abrir `MyPME_Investor_Deck.html` num browser para a apresentacao principal.
2. Usar `01_Resumo_Executivo.md` como one-pager de apoio.
3. Levar `03_Ficha_Tecnica_do_Produto.md` para perguntas tecnicas e due diligence inicial.
