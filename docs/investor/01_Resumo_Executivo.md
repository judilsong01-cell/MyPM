# MyPME - Resumo Executivo

## O que e

MyPME e uma aplicacao mobile para micro e pequenas empresas em Angola controlarem vendas, despesas, fiado e operacao diaria, mesmo em contextos com internet instavel.

## Problema

Milhares de negocios pequenos ainda operam com caderno, WhatsApp e memoria. Isso cria perda de receita, falta de visibilidade financeira, fiado sem controlo e ausencia de historico confiavel para tomada de decisao.

## Solucao

MyPME resolve o problema com uma experiencia mobile simples, orientada a uso no balcao:

- registo rapido de vendas e despesas
- controlo de fiado
- operacao offline-first
- sincronizacao cloud quando existir internet
- autenticacao por utilizador
- base pronta para evolucao para servicos fintech

## Porque agora

- a penetracao de smartphones continua a crescer
- pequenos negocios querem digitalizar, mas nao toleram produtos complexos
- a instabilidade de rede em varios contextos torna apps 100% online menos praticas
- a proximidade entre operacao, historico financeiro e pagamentos cria uma ponte natural para servicos de maior valor

## Diferenciais

- offline-first real, nao apenas cache superficial
- experiencia pensada para rapidez operacional
- produto desenhado para Angola, moeda local e contexto real
- caminho claro para evolucao em direcao a pagamentos, score e integracoes bancarias

## Estado atual

- app Android funcional
- branding e identidade MyPME aplicados
- autenticacao por email
- SQLite local com sincronizacao preparada
- dashboard, registo, fiado e inventario em evolucao
- backend fintech modular iniciado em Node.js + Prisma + PostgreSQL

## Oportunidade

MyPME pode começar como software operacional para pequenos negocios e evoluir para plataforma de dados financeiros, pagamentos e servicos de valor acrescentado para um segmento historicamente subatendido.
