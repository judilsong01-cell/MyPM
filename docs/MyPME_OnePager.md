# MyPME - One Pager

## Proposta

MyPME e um sistema mobile offline-first para pequenos negocios em Angola controlarem vendas, despesas, fiado e operacao do dia a dia com rapidez e seguranca.

## Problema

Pequenos negocios continuam a operar com registos manuais, fiado desorganizado e pouca visibilidade financeira. A internet instavel torna muitas solucoes 100% online pouco praticas para uso real.

## Solucao

- registo rapido de entradas e saidas
- controlo de fiado
- base local com funcionamento offline
- autenticacao por utilizador
- sincronizacao cloud quando houver internet
- estrutura pronta para evolucao para servicos fintech

## Estado atual

- app Android funcional
- branding MyPME aplicado
- dashboard, vendas, despesas e fiado operacionais
- inventario com base tecnica pronta e camada visual em consolidacao
- backend fintech modular iniciado

## Diferenciais

- offline-first real
- experiencia pensada para operacao em balcao
- seguranca local ao trocar de conta
- base de dados historica que viabiliza scoring, pagamentos e servicos financeiros

## Modelo de negocio

- freemium para adocao inicial
- premium para sync, relatorios e multi-dispositivo
- camada futura de pagamentos e produtos financeiros

## Proximo passo

Transformar o MVP funcional numa versao comercial robusta, validar pilotos e preparar a camada fintech.
